=============================================================
СТРАТЕГИЯ STALE DETECTION ДЛЯ CADREMPLOI
=============================================================

Нет абсолютной даты → используем cache:
  1. Первый скрап: сохранить jobId + firstSeen = NOW()
  2. Следующий скрап: если jobId ещё в результатах → обновить lastSeen
  3. Если lastSeen - firstSeen >= 60 дней → SIGNAL_STALE
  4. Если jobId исчез из результатов → filled or expired, удалить из cache

Поле firstSeenAt в данных актора = дата которую видит САМА БАЗА Cadremploi (не надёжно).
Используй НАШЕ firstSeen из локального cache.json.
