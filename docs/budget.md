## Вариант A: Exa Monitors вместо ручного скана

Заменяем ручной запуск exa_scanner.py на Monitors с webhook → n8n.

| Компонент | Цена/единица | Объём/неделю | $/неделю | $/месяц |
|-----------|-------------|-------------|---------|---------|
| Exa Monitors (новостные сигналы) | $15/1k req | 40 запросов | $0.60 | $2.40 |
| Apify Indeed scraper | $6/1k results | 300 результатов | $1.80 | $7.20 |
| Apify LinkedIn scraper | $1/1k results | 500 результатов | $0.50 | $2.00 |
| Apify free tier | — | — | — | -$10.00 |
| **ИТОГО нетто** | | | **$2.90** | **~$1.60–3** |

Что меняется:
- +$0.32/неделю (+$1.28/месяц) за автоматизацию Exa
- Встроенная дедупликация (не нужен свой код)
- Webhook → n8n: сигналы приходят сами без ручного запуска
- outputSchema → структурированный JSON без постобработки

## Вариант B: Exa Monitors вместо Apify job boards

Exa Monitors ищет "food exec job postings Germany/France/NL" по вебу вместо прямого скрапинга Indeed/LinkedIn.

| Компонент | Цена/единица | Объём/неделю | $/неделю | $/месяц |
|-----------|-------------|-------------|---------|---------|
| Exa Monitors (новости + job boards) | $15/1k req | 80 запросов | $1.20 | $4.80 |
| Apify free tier | — | — | — | — |
| **ИТОГО нетто** | | | **$1.20** | **$4.80** |

Плюсы: дешевле Apify, шире покрытие (не только Indeed/LinkedIn)
Минусы: нет posting date, нет isExpired флага, менее структурировано → нужен outputSchema + доп. фильтрация
Статус: ТРЕБУЕТ ТЕСТА — неизвестно, насколько хорошо Exa находит job postings vs Apify

## Рекомендация

1. **Сначала тест B** — запустить Exa Monitors на 10 job-board запросов за 1 неделю (~$0.15), сравнить покрытие vs Apify Indeed результатов из сессии 2 (349 результатов → 8 компаний).
2. **Если B работает** → переходим на Вариант C: $5/месяц, полная автоматизация, без Apify.
3. **Если B не работает** → Вариант A: $3/месяц, только Exa автоматизировано, Apify остаётся.

Экономия Apify free tier освобождает $10/месяц под другие нужды если от Apify уходим.
